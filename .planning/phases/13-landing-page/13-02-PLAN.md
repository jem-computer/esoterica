  <action>
Replace the placeholder index.html with the complete landing page implementation.

**HTML Structure:**
- Semantic HTML5 with data-theme attribute on html element
- Meta tags: charset, viewport, description, Open Graph, Twitter Card
- Favicon links: SVG (modern), ICO fallback (if needed), Apple Touch Icon
- Google Fonts: Cormorant Garamond (400, 500, 700), Source Sans 3 (400, 600), JetBrains Mono (400)
- Inline theme detection script in head (prevents FOUC)
- Inline CSS in style tag (under 10KB for performance)
- Inline JS at end of body

**Page Layout:**
- Header: minimal nav with "Esoterica" text-logo left, GitHub icon link right, theme toggle button
- Hero section (100svh height):
  - Full-bleed background image (hero-primary.png)
  - Dark overlay (CSS ::before pseudo-element, opacity varies by theme)
  - Centered content:
    - H1: "Ancient patterns, new paths" (tagline - primary headline)
    - H2: "Esoterica" (name - secondary)
    - Brief value prop paragraph
    - Install command with copy button inline
- Footer: minimal with GitHub link and copyright

**CSS Implementation:**
Use colors from brand guide:
- Primary: #e5c8bc (hero text, CTAs)
- Secondary: #4b3829 (subheadings, dark mode text)
- Accent: #a7acaf (links, hover states)
- Warm Neutral: #9b7c6c (backgrounds, button hover)
- Cool Neutral: #9497a1 (disabled states)
- Dark Base: #7a7b7b (body text)

Light theme overlay: rgba(0, 0, 0, 0.25-0.35) - tune for contrast
Dark theme overlay: rgba(0, 0, 0, 0.5-0.6) - darker for legibility

Typography:
- H1: Cormorant Garamond, 3rem mobile / 5rem desktop, Medium (500)
- H2: Cormorant Garamond, 2.25rem mobile / 3rem desktop, Regular (400)
- Body: Source Sans 3, 1rem, Regular (400)
- Code/command: JetBrains Mono, 0.875rem

Responsive breakpoints:
- Mobile-first base styles
- 768px: Tablet enhancements
- 1024px: Desktop enhancements

**JavaScript Implementation:**
1. Theme toggle:
   - Click handler toggles data-theme between light/dark
   - Saves preference to localStorage
   - Updates aria-label for accessibility

2. System preference listener:
   - Listens for prefers-color-scheme changes
   - Only applies if no localStorage preference set

3. Copy to clipboard:
   - Uses navigator.clipboard.writeText()
   - Changes button text to "Copied!" for 2 seconds
   - Fallback for non-HTTPS environments (though GitHub Pages is HTTPS)

**Install command:** `npm install -g esoterica` (verify this is correct install method)

**Meta tags for social sharing:**
- og:title, og:description, og:image (og-image.png)
- twitter:card (summary_large_image), twitter:image
- URL: https://111ecosystem.github.io/esoterica/ (GitHub Pages project site format)

**Accessibility:**
- Proper heading hierarchy (H1 -> H2)
- aria-labels on buttons
- Focus visible states
- Color contrast meeting WCAG AA (4.5:1)
